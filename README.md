Description du projet
Le projet Mini Trello est une application web de gestion de tâches inspirée de la plateforme Trello. Il permet aux utilisateurs d’organiser leurs projets à l’aide de tableaux (boards), de colonnes et de cartes (tâches), en suivant le principe de la méthode Kanban.
L’objectif est de faciliter la gestion du travail individuel et collaboratif à travers une interface simple, intuitive et moderne.
Objectifs :
	Organiser les tâches de manière efficace
	Suivre l’avancement des projets
	Faciliter la collaboration entre utilisateurs
	Appliquer les concepts de génie logiciel (UML, architecture, etc.)
	Mettre en pratique les technologies web modernes

 Technologies utilisées
Frontend : React, HTML, CSS, Bootstrap
Backend : Laravel (PHP)
Base de données : MySQL
Serveur local : XAMPP
Outils : VS Code, GitHub
Fonctionnalités :
 Côté utilisateur :
Création de boards (projets)
Ajout de colonnes (To Do, Doing, Done)
Gestion des tâches (ajout, modification, suppression)
Déplacement des tâches (drag and drop)
Ajout de commentaires et fichiers
Invitation de membres (collaboration)
Côté administrateur :
Gestion des utilisateurs
Consultation des boards et tâches
Suppression des données si nécessaire
Suivi global du système
